# Mission & Definition of Done (정본 완성 기준)

Created: 2026-07-05 (KST)
Owner intent (사용자 진술 요약): 양산 휴머노이드/협동로봇을 곧 일반인
(초보자/비전공자/대학생)이 접하게 된다. 그들이 (1) 환경·상황에 맞는 제어
방식을 이해하고 선택하며, (2) 채택한 방식의 파라미터를 어떤 원리로 설정해야
하는지 이해할 수 있어야 한다. 전공자도 임피던스 설정(강성·토크)을 잘못 잡아
사물을 파손하는 일이 흔하다. 따라서 이론·역사·배경부터 확실하게 이해시키는,
유도 생략과 표기 불일치가 없는 **신뢰 가능한 초보자용 튜토리얼/리뷰**를
만든다. 실물 로봇이 없으므로 시뮬레이터+이론 중심으로 하고, 실물 데모
(유튜브)·실측 데이터/코드 공유는 추후(2027+) 로드맵으로 미룬다.

핵심 차별점 (기존 논문 대비):
1. 유도 과정 무생략 (High/Medium 격차 0 달성, 앵커 113개로 게이트됨)
2. 일관된 표기 (부록 A 표기 체크리스트)
3. 이론 ↔ 재현 가능한 시뮬레이터 실험의 1:1 연결
4. **상황별 제어 방식 선택**과 **안전한 파라미터 설정 원리** (신규 보강 대상)
