# Educator guide

Each experiment follows the same sequence: goal, prediction, manipulation, observation, and review. Ask learners to change one variable at a time and explain the priority plot before opening advanced metrics.

Use YAML configs for prepared demonstrations and comparison batches for class discussion. Saved `manifest.json` files provide the scenario, resolved config, seed, runtime, model license, and hashes needed to audit a submission.

Recommended classroom evidence is one learner-control action, one prediction-backed observation, the priority plot, and a short explanation of the visible effect.
